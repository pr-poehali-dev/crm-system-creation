# Как подключить диалоги из Avito к CRM

## Что нужно сделать:

### 1. Получить API ключи от Avito

Avito предоставляет API для работы с объявлениями и сообщениями:

**Вариант А: Официальный Avito API (для бизнеса)**
1. Перейдите на https://www.avito.ru/professionals/api
2. Зарегистрируйтесь как бизнес-пользователь
3. Подайте заявку на доступ к API
4. После одобрения получите:
   - Client ID
   - Client Secret
   - Access Token

**Вариант Б: Avito Автозагрузка (упрощённый)**
1. Зайдите в личный кабинет Avito
2. Раздел "Настройки" → "API и интеграции"
3. Создайте приложение
4. Скопируйте учётные данные

### 2. Внести данные в CRM

1. Откройте CRM → раздел **"Интеграции"**
2. Найдите карточку **"Avito"**
3. Нажмите **"Настроить"**
4. Вкладка **"Доступы"**:
   - Вставьте **Client ID**
   - Вставьте **Client Secret**  
   - Вставьте **User ID** (ваш ID на Avito)
5. Нажмите **"Сохранить"**
6. Переключите тумблер **"Включить интеграцию"**

### 3. Загрузить диалоги

1. Перейдите в раздел **"Лиды"**
2. Нажмите кнопку **"Загрузить из Avito"**
3. Все новые сообщения от клиентов загрузятся автоматически

### 4. Работа с диалогами

Каждое новое сообщение с Avito:
- ✅ Создаёт новый лид
- ✅ Показывает имя клиента, телефон (если указан)
- ✅ Сохраняет текст сообщения
- ✅ Отображает интересующий товар (если указан)
- ✅ Позволяет ответить прямо из CRM

## Важно знать:

⚠️ **Лимиты Avito API:**
- Бесплатный тариф: до 1000 запросов/день
- Платный: от 10,000 запросов/день

⚠️ **Что CRM НЕ ДЕЛАЕТ автоматически:**
- Не публикует объявления на Avito
- Не обновляет цены в объявлениях
- Не удаляет объявления

✅ **Что CRM ДЕЛАЕТ:**
- Загружает новые сообщения от клиентов
- Создаёт лиды для каждого обращения
- Позволяет вести воронку продаж
- Сохраняет историю общения

## Если возникли проблемы:

**"Ошибка загрузки":**
- Проверьте правильность Client ID и Secret
- Убедитесь, что токен не истёк
- Проверьте лимиты API

**"Нет новых диалогов":**
- Убедитесь, что у вас есть новые сообщения на Avito
- Проверьте, что интеграция включена
- Попробуйте обновить токен

**"Не могу ответить клиенту":**
- Сейчас ответы доступны только через Avito
- В будущих версиях добавим отправку ответов через API

## Альтернативный вариант (без API):

Если у вас нет доступа к API Avito:
1. Добавляйте лиды вручную кнопкой "Добавить лид"
2. Копируйте информацию из Avito в CRM
3. Ведите клиента по воронке продаж внутри CRM
